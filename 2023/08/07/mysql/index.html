数据库

数据库就是存储数据库的仓库，数据库是一个以某种有结构的有顺序存储的数据集合

数据库相关概念

名称 全称 简称
数据库 存储数据的仓库，数据是有组织的进行存储 DataBase (DB)
数据库管理系统 操纵和管理数据库的大型软件 DataBase Management System (DBMS)
SQL 操作关系型数据库的编程语言，定义了一套操作关系型数据库统一标准 Structured Query Languge (SQL)

关系型数据库

安全系数高，容易理解，比较浪费磁盘空间

  • Oracle：大型的收费数据库
  • MySQL：开源免费的中小型数据库
  • SQL Server：Microsoft公司推出的收费的中兴数据库
  • PostgreSQL：开源免费的中小型数据库
  • DB2：IBM公司的大型收费数据库产品
  • SQLLite：嵌入式的卫微型据库
  • MariaDB：开源免费的中小型数据库

非关系型数据库

安全系数低，效率高

典型代表有：NoSQL、BigTable (Google)、Cassandra、MongoDB、CouchDB

还包括键值数据库：Apache、Cassandra (Facebook)、LevelDB (Google)

区别

  • 关系型数据库：实例->数据库->表->记录行、数据字段
  • 非关系型数据库：实例->数据库->集合->键值对

数据库端口号

关系型数据库 MySQL（3306）、Oracle（1521）、Sql Server（1433）、DB2（5000）、PostgreSQL（5432）、DM达梦（5236）
非关系型数据库 Redis（6379）、Memcached（11211）、MongoDB（27017）

使用MySQL提供的客户端命令行工具

使用系统自带的命令行工具

  • 查看当前用户所有数据库show databases;

  • 查看当前数据库内所有表show tables;

SQL基本语句

  • create database 库名 //创建数据库
  • create table 表名 //创建新表

  • drop table 表名 //删除表
  • select 列名 from 表名 //查看表中某一列的信息

  • select distinct 列名 from 表名 //查看表中某一列的信息并过滤重复的值
