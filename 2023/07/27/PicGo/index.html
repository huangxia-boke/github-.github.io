typora+PicGo

介绍

1.typora是一个Markdown格式的编辑器，可对博客文章进行编辑

2.PicGo是一个自动上传图片到图床的工具

参考链接

1.下载typora

2.下载PicGo

3.PicGo手册

1.下载node.js

搭建

3.typora-偏好设置

  • 勾选“对本地位置的图片应用上述规则；对网络位置的图片应用上述规则”
  • 上传服务选择“PicGo(app)”

4.PicGo设置

  • 在显示的图床中只保留“GitHub”

  • 点击图床设置，新建图床，设置

5.测试typora

  • 图片上传成功

  • 环境搭建完成!

新建博客文章

相关命令

  • hexo new “我的博客” #新建文章
  • hexo new page “第一页” #新建页面
  • hexo generate #生成
  • hexo server #启动服务预览
  • hexo deploy #部署
  • hexo clean #清除缓存
  • hexo version #查看版本
  • hexo server -s #静态模式
  • hexo server -p 50000 #更改端口
  • hexo server -i 192.168.1.1 #自定义IP
  • hexo server -generate #生成并本地预览
  • hexo d -generate #生成并上传
  • 可进入目录下

  • .md后缀的是你创建的文章
  • 右击“Typora”方式打开
  • 接下来就可以进行文章的编写啦！
